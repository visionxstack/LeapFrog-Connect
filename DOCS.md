# Leapfrog Connect Integrated LMS & HR Platform

## Project Overview
Leapfrog Connect Pvt. Ltd. aims to build a unified web ecosystem bridging the gap between education and employment in Nepal. The platform integrates a Learning Management System (LMS) with an HR Service Tracking Platform to streamline training, event management, and candidate placement for students, professionals, and partner employers.

## 1. Project Goals & Audience
● Goal: Create a scalable "Skills-to-Jobs" pipeline.

● Target Roles:
○ Admins/Staff: Manage courses, events, finances, and recruitment funnels.
○ Prospects: Discover learning, build digital profiles, and apply for jobs.
○ Partner Companies: Access vetted talent, manage interviews, and hire.

## 2. Our Main Objectives
● Students profile -> Their skill track / journey track
● Exams and certifications
● Talent pool for companies (Good recruitment funnel)
● Dashboards for data-driven analytics

## 3. Core Functional Modules
● LMS (Training Management): Course creation with multimedia support, live class integration (Zoom/Meet), assessment engines (quizzes/tests), and automated certification.

● HR & Placement: A central candidate database with customizable recruitment funnels (Applied → Interview → Hired) and employer feedback loops.

● Event Management: End-to-end handling of hackathons and workshops, including registration, waitlisting, and payment.

● Personalized Dashboards: Role-specific views for tracking progress, interview schedules, and placement metrics.

## 4. Key Integrations & Technical Specs (Optional)
● Payments: Local gateways (eSewa, Khalti, Fonepay) plus international options (Stripe).

● Communication: Automated SMS (Sparrow SMS/Ncell) and Email (SendGrid/Mailchimp) for reminders and alerts.

● Scheduling: Two-way sync with Google and Outlook Calendars for interviews.

● Architecture: Responsive web design with cloud hosting (AWS/GCP), focusing on OWASP security standards and data privacy.

## 5. User Experience & Reporting
● For Candidates: Relevant profile creation, journey tracking, profile sharing via public links, resume assistance, and real-time application tracking.

● For Employers: Advanced filtering of the talent pool based on training performance and skills.

● Analytics: Comprehensive reporting on enrollment trends, revenue reconciliation, and placement success rates.

## 6. Technical Considerations
● Preference for specific technologies (e.g., backend language like Python/Django, Node.js/Express, PHP/Laravel; frontend framework like React, Vue, Angular)?

● Database preference (e.g., PostgreSQL, MySQL)?

● Cloud hosting preference (AWS, GCP, Azure)?

● Adherence to specific coding standards or methodologies (e.g., Agile)?

## 7. Non-Functional Requirements & Deliverables
● Performance: High availability (99.5% uptime) with page loads under 3 seconds.

● Deliverables: Full source code, technical documentation, user manuals, and staff training.

## Success Criteria
A secure, localized, and high-performance platform that successfully converts Leapfrog trainees into employed professionals through a seamless digital experience.
