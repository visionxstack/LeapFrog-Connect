### 6. TECHNICAL RULES

- Pure HTML, CSS, JavaScript only — no frameworks, no libraries except:
  - Font Awesome CDN for icons (already likely in your pages)
  - Google Fonts CDN (Playfair Display / Nunito or whatever your pages use)
- Use `<script type="module">` or plain script tags with defer
- All JS files loaded via `<script src="../js/filename.js" defer>` in each HTML file
- Use `DOMContentLoaded` event listener in every page JS file
- No alert() or confirm() — use custom modals and toasts instead
- All localStorage reads must have null-safety (check if value exists before parsing)
- URL params: use `new URLSearchParams(window.location.search)` to read ?id= values
